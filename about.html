PkéAPI

About

This website allows you to get data of a Pokémon you search. You can find their Pokédex number, type, abilities, base stats, and moves. This website offers a better user experience for someone with prior knowledge of the Pokémon video games, as the PokéAPI is made based on them. You can start by searching Pikachu!

If you want to see the code check my GitHub.

How to use?

Type a Pokémon name and click the search button or click enter.

Make sure you spell the name correctly or you will NOT get any data. Some Pokémon are not available through this API (Application Programming Interface). See below in the Unavailable Pokémon section.

Finding Alternate Forms

To find alternate forms you MUST use hyphen ( - ). For example Charizard-Mega-X.

To find Nidoran♂Male and Nidoran♀Female add a hyphen and the first letter of the gender. Type Nidoran-M or Nidoran-F. They are special Pokémon who share the same name, but different gender.
